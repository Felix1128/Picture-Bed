# Open source Picture bed

出于自用的目的，又找不到Java写的开源的程序，然后使用开源上传组件[ bootstrap-fileinput ](https://github.com/kartik-v/bootstrap-fileinput/)用 Spring Boot 写了一个图床

> 目前支持：七牛云，阿里云


### Configuration

> #### 使用的时候需要按照`application.properties` 里面的注释配置即可
